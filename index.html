단어 보기
단계를 선택하면 해당 단계의 단어 카드가 표시돼요. 복수 선택도 가능해요.

상단의 [발음 문제] / [뜻 문제] 버튼을 누르면 현재 선택된 단계의 단어들로 무한 랜덤 문제가 팝업으로 시작돼요.
한자 탭에서는 [객관식] / [주관식]으로 표시돼요.

팝업 안에서 모드 전환(발음 ↔ 뜻)이 가능하고, 틀린 문제는 [오답 저장]으로 즉시 저장할 수 있어요.

[단어선택] 버튼으로 선택 모드를 켜면 카드를 직접 골라 문제 세트를 만들 수 있어요.
첫 카드 클릭 후 Shift + 클릭으로 범위 선택도 가능해요.
선택 후 하단 바의 [문제풀기 ▲]를 누르면 선택된 단어들로 유한 모드(한 번씩 전부 출제, 진행률 바 포함)가 시작돼요.
유한 모드 완료 후 결과 팝업에서 오답/정답 탭 전환, 단어 선택 후 태그 변경, 오답 다시 풀기가 가능해요.
단어 태그
각 단어에 미분류 · 학습중 · 복습중 · 완료 태그를 붙여 학습 상태를 관리할 수 있어요.

태그 붙이기 — [단어선택] 모드에서 단어를 선택한 뒤 하단 바의 [태그변경 ▲]에서 원하는 태그를 선택해요.
문제 결과 팝업 안에서도 단어를 선택해 태그를 바꿀 수 있어요.

태그 표시 — 상단의 [태그 표시] 버튼을 켜면 카드에 태그 배지가 나타나고, 태그별 필터 버튼이 표시돼요.
태그 데이터는 브라우저에 저장되어 창을 닫아도 유지돼요.
동사 변형
일본어 동사의 활용형을 연습하는 문제예요. 세 가지 유형이 있어요.

문법 주어진 활용형이 어떤 변형인지 고르는 객관식
변형 객관식 지정된 변형 형태를 보기에서 고르기
변형 주관식 지정된 변형 형태를 직접 히라가나로 입력

화면 상단에서 1그룹 / 2그룹 / 3그룹을 선택해 원하는 그룹만 출제할 수 있어요.
그 아래 유형 선택에서 て형, た형, ない형 등 연습할 활용형을 고를 수 있어요.
받아쓰기
음성을 듣고 히라가나로 받아쓰는 문제예요. 띄어쓰기·문장부호 없이 입력해요.
▶ 다시 듣기 버튼으로 반복 재생할 수 있고, 느리게 버튼으로 천천히 들을 수 있어요.
회차와 번호를 선택해 원하는 범위만 출제 가능해요.
한바퀴 돌기
동사 변형·받아쓰기 문제 카드 우상단의 [한바퀴 돌기] 버튼을 누르면 현재 선택된 범위의 문제를 셔플 후 한 번씩 전부 출제해요.
한바퀴 중에는 정답 여부가 즉시 표시되지 않고 빠르게 넘어가요.
완료되면 결과 팝업에서 점수와 오답 목록을 확인하고, 오답이 있으면 [오답 다시 풀기]로 바로 재도전할 수 있어요.
레벨·그룹 변경, 탭 이동, 버튼 재클릭 시 취소돼요.
오답 복습
문제를 틀렸을 때 [오답 저장] 버튼을 눌러 저장하면, 상단바의 오답 복습 탭에서 모아볼 수 있어요.
한바퀴 돌기 결과 팝업에서도 오답을 한꺼번에 저장할 수 있어요.

오답 카테고리를 클릭하면 저장된 오답 목록이 표시돼요.
[복습하기]를 누르면 해당 오답들만 무작위로 출제돼요. 같은 문제를 3번 맞히면 오답 횟수가 1 차감돼요.
휴지통 버튼으로 삭제 모드를 켜면 오답 횟수를 직접 조정하거나 전체 삭제할 수 있어요.
오답 데이터는 브라우저에 저장되어 창을 닫아도 유지돼요.
PNG 저장 · 인쇄
[단어선택] 모드에서 단어를 골라 [PNG 저장] 또는 [인쇄/PDF] 버튼으로 내보낼 수 있어요.

PNG 저장 — 선택한 단어를 A4 4×8 격자 이미지로 저장해요. 32개 초과 시 압축 파일(.zip)로 다운로드돼요.
인쇄/PDF — 브라우저 인쇄 화면이 열려요. 프린터 대신 PDF로 저장을 선택하면 PDF로 내보낼 수 있어요.

버그 수정
· 외래어 문제 — 단어/뜻 모드 라벨이 반대로 표시되던 문제 수정. 오답 복습도 동일하게 수정.
· 한바퀴 돌기 — 일반 모드에서 틀린 후 한바퀴 시작 시 오답 저장 버튼이 남아있던 문제 수정.
· 오답 복습 — 주관식 정답 제출 후 엔터로 다음 문제 이동 안 되던 문제 수정.
· 모바일 입력 — 소프트키보드 완료 버튼으로 주관식 제출 안 되던 문제 수정.

기능 추가 · 변경
· 단어 보기 대개편 — [단어선택] [발음문제] [뜻문제] 버튼 추가. 무한모드(완전 랜덤) 및 유한모드(선택 단어 한바퀴) 팝업 퀴즈 구현.
· 단어 태그 — 미분류·학습중·복습중·완료 태그 시스템 추가. 카드 배지 표시, 태그별 필터, localStorage 저장.
· 결과 팝업 — 오답/정답 탭 전환, 단어 선택 후 태그 변경, 오답 다시 풀기 기능 추가.
· 한바퀴 결과 — 오답 다시 풀기 버튼 추가 (동사변형·받아쓰기 포함).
· 단어 문제 탭 제거 — 단어 보기 탭의 문제풀기 기능으로 완전 대체.
· 팝업 키보드 — 단어보기 퀴즈 팝업(객관식 숫자키·Enter·→) 키보드 지원 추가.
· 결과 팝업 Shift 선택 — 선택 모드에서 Shift+클릭 범위 선택 지원.
· ESC로 팝업 닫기 — 전체 팝업(퀴즈, 결과, 한바퀴, 복습, PNG확인) ESC 키 지원.
